How to setup this system in your server?

1. copy all files to your desire location.
Suggested to create a directory called 'library'

2. open file 'mysqlConnectInfo.php' and edit the following lines:
$mysqlHost = "localhost"; --> change to your mySQL host name
$mysqlLogin = "root"; --> change to your mySQL login name
$mysqlPassword = "admintoni"; --> change to your mySQL login password
$dbname = "onlinelib"; --> change to your desire database name

3. Load page 'checkDatabase.php' from your browser.
or simply click here to get in that page.
